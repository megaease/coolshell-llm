Question: Web在线WYSIWYG编辑器
作者：陈皓，左耳朵耗子
1. FCKeditor 这些在线编辑器中最著名的一个，其功能相当的强大，很像一个Web的Word软件。
2. NicEdit 是一个基于Javascript 编辑器，它可以很容易地被集成到任意的网页中。
3. TinyMCE 是另一个很有名的所见即所得的编辑器，其受LGPL license控制。
4. jwysiwyg 是一个基于 jQuery 的WYSIWYG 插件，相当小，只有7kb的大小，而且相当的简单易用。
5. Yahoo UI Library: Rich Text Editor 这个富文本编辑器是 Yahoo YUI 库中的一部分，用户或以非常简单的扩展它。
6. Xinha 也是一个相当强大的WYSIWYG HTML 编辑器，它可以兼容于所有的浏览器，并被开源社区所支持。
7. Openwysiwyg 也是另一个开源的跨浏览器的 WYSIWYG 编辑器，别看他外表长得不怎么样，但他有很多的功能，特别是表格编辑的功能。

Question: 请推荐一些Web在线WYSIWYG编辑器
作者：陈皓，左耳朵耗子
8. Free Rich Text Editor 是一个超级简单并且是免费的WYSIWYG 编辑器，它非常容易用来实现和管理基于XHTML的文本。
9. WMD: The Wysiwym Markdown Editor 是一个简单和轻量级的编辑器，它主要用于Blog的评论系统或是论坛回贴系统。
10. TTW HTML Editor 也是一个很简单轻量级的WYSIWYG编辑器，其主要由Javascripts编写，其拼写检查由SpellerPages编写。
11. Free Text Box 也是一个很常用的HTML editor，只不过它只支持ASP.NET。
12. WYMeditor 是一个 XHTML 的编辑器。WYMeditor 可以创建并生成非常完美的XHTML 结构的源码，并完全严格遵守W3C XHTML 规范。
13. BlueShoes Wysiwyg Editor 这是一个DHTML 和Javascript 开发的编辑器，它有很多非常酷的功能。
14. markItUp 是一个jQuery 的JavaScript 插件。它非常的轻量，可以非常容易的定制。
15. SPAW Editor 这个WYSIWYG 编辑器是一个多页的编辑器，浮动式的工具条和很酷的用户接口，目前只有PHP 和.NET 版本。
